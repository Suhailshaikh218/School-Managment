# School Management System

A comprehensive web-based school management system built with HTML, CSS, and JavaScript.

## Features

- **Student Management**: Add, edit, delete, and view student records
- **Teacher Management**: Manage teacher profiles and information
- **Course Management**: Create and assign courses to teachers
- **Attendance Tracking**: Mark and monitor daily attendance
- **Grade Management**: Track student grades by course and exam type
- **Schedule Management**: Organize class schedules by day
- **Fee Management**: Track fee payments and outstanding amounts
- **Dashboard**: Overview of key metrics and recent activities

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage for data persistence

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Suhailshaikh218/School-Managment.git
   ```

2. Navigate to the project directory:
   ```bash
   cd School-Managment
   ```

3. Open `index.html` in your preferred web browser
